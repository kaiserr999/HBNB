# HBNB

A Python-based AirBnB clone project.

## Description

HBNB is a simplified clone of the AirBnB web application. This project implements core models and a command-line interpreter to manage application objects.

## Models

- **BaseModel** — Foundation class providing common attributes (`id`, `created_at`, `updated_at`) and methods for all models.
- **User** — Represents a registered user with attributes such as email, password, first name, and last name.

## Usage

```python
from user import User
from basemodel import BaseModel
```
